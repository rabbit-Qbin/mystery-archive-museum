# 部署指南

## 本地开发

### 1. 安装依赖

```bash
cd website
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:5173`

### 3. 构建生产版本

```bash
npm run build
```

## 部署到 Vercel

### 方法 A: 通过 GitHub (推荐)

1. 将代码推送到 GitHub
2. 访问 [Vercel.com](https://vercel.com) 并登录
3. 点击 "Add New..." -> "Project"
4. 导入你的 GitHub 仓库
5. 设置根目录为 `website`
6. 框架选择 "Vite"
7. 点击 "Deploy"

### 方法 B: 命令行部署

```bash
cd website
npx vercel
```

## 注意事项

- 部署时根目录需要设置为 `website/`
- 确保 archives/ 文件夹中的档案已正确注册到 registry.js
- 评论数据存储在用户浏览器的 LocalStorage 中，不会上传到服务器
