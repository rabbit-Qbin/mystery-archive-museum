# 档案分类标准

## 分类编号规则

| 分类 | 编号前缀 | 文件夹 | 说明 |
|------|----------|--------|------|
| 未知生物 | CRE-xxx | creatures/ | 异常生物、神秘物种 |
| 禁忌地带 | LOC-xxx | locations/ | 危险区域、异常地点 |
| 诡秘物品 | ART-xxx | artifacts/ | 异常物品、神秘器物 |
| 未解事件 | EVT-xxx | events/ | 神秘事件、未解之谜 |

## 档案文件格式

```javascript
export default {
  id: "XXX-001",           // 档案编号
  title: "档案标题",        // 档案名称
  category: "creature",    // 分类ID: creature/location/artifact/event
  tags: ["#标签1", "#标签2"], // 标签数组 (3-4个)
  summary: "档案摘要...",   // 简要描述
};
```

## 命名规范

- 文件名格式: `编号-中文名称.js`
- 例: `CRE-001-深海美人鱼.js`
- 编号从 001 开始递增
- 同一分类内编号不可重复
