## 档案模板

创建新档案时，复制以下模板：

```javascript
export default {
  id: "XXX-00X",
  title: "档案标题",
  category: "creature",  // creature/location/artifact/event
  tags: ["#标签1", "#标签2", "#标签3"],
  summary: "档案摘要，简要描述档案内容...",
};
```

---

## 添加档案步骤

1. **创建档案文件**
   ```
   archives/[分类]/[编号]-[名称].js
   ```

2. **编写档案内容**
   - 使用上面的模板
   - 填写必填字段: id, title, category, tags, summary
   - 可选字段: discoveryDate, location, threatLevel

3. **注册档案**
   - 打开 `archives/registry.js`
   - 在对应分类下添加一行:
   ```javascript
   { file: './[分类]/[编号]-[名称].js', id: '[编号]' },
   ```

4. **验证**
   - 启动本地服务: `npm run dev`
   - 检查新档案是否显示
