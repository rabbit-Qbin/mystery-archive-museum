# 快速开始指南

## 项目启动

### 1. 安装依赖 (首次)

```bash
cd website
npm install
```

### 2. 启动开发服务器

```bash
cd website
npm run dev
```

访问 `http://localhost:5173`

### 3. 构建生产版本

```bash
cd website
npm run build
```

---

## 添加新档案 (3步)

### 步骤1: 创建档案文件

在对应分类文件夹创建新文件:

```
archives/creatures/CRE-003-新生物.js
```

### 步骤2: 编写档案内容

```javascript
export default {
  id: "CRE-003",
  title: "新生物名称",
  category: "creature",
  tags: ["#标签1", "#标签2"],
  summary: "档案摘要...",
};
```

### 步骤3: 注册档案

打开 `archives/registry.js`，在对应分类下添加:

```javascript
{ file: './creatures/CRE-003-新生物.js', id: 'CRE-003' },
```

完成! 刷新网站即可看到新档案。

## 常见问题

**Q: 新档案添加后没有显示?**
A: 检查是否在 `registry.js` 中注册了档案。

**Q: 如何修改分类名称?**
A: 编辑 `website/src/data/categories.js`

**Q: 评论数据存在哪里?**
A: 存储在浏览器 LocalStorage 中，清除缓存会丢失。

**Q: 如何部署到线上?**
A: 参考 `docs/部署指南.md`
